# najathakram.info

Personal site of Najath Akram, Ph.D. — signal processing engineer working on
5G, 6G, and O-RAN physical-layer systems.

Built with Next.js 16, TypeScript, Tailwind v4, and Motion. Deployed on Vercel.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
